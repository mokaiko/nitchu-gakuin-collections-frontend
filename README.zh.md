## 概述

本网站用于查看与领取日中学院发行的数字藏品（NFT）。  
管理员可通过后台管理界面配置合约参数并管理各类 NFT。  
网站通过 MetaMask 浏览器扩展插件进行连接。

---

## 使用前准备

1. 安装 MetaMask（Chrome 浏览器推荐）。
2. 创建或导入钱包。
3. 确认页面中显示的合约地址是否正确。
4. 在 MetaMask 中切换到 **Optimism 主网**。

---

## 普通用户（查看与领取）

### 1. 连接钱包

点击页面顶部的“MetaMask 接続”按钮。

### 2. **藏品列表**

可查看所有已发行藏品。  
若能正常显示图片，则说明 SVG 已成功上传。

### 3. **领取页面**

显示满足以下条件的藏品：

- 当前处于激活状态
- SVG 上传已最终锁定

#### 若按钮不可点击，常见原因包括：

- 钱包未连接 / 网络不匹配
- 达到供应上限
- 不在白名单中
- 已领取过

满足条件后点击“受け取る”，在 MetaMask 中确认，即可领取 **1 枚 NFT**。

## 管理员（Owner 或具有管理员权限的地址）

### 1. 进入管理界面

使用 Owner 或管理员地址连接后，会显示“管理”标签页。

### 2. 管理功能说明

#### **合约信息**

显示合约地址、Owner、合约版本。

#### **创建新藏品**

可设置：

- 名称
- 简介
- 最大供应量（0 表示无限制）
- 是否启用白名单
- 是否在创建后立即激活

#### **SVG 上传（分片方式）**

流程：

1. 输入 Token ID、chunkIndex（从 0 开始）、粘贴 SVG 文本（建议 ≤30KB）。
2. 点击 **步骤 1：追加分片**。
3. 全部分片上传完成后，点击 **步骤 2：完成上传（锁定不可修改）**。

#### **状态更新**

可为每个 Token ID 切换：

- 白名单开/关
- 激活/停用

当前状态显示在页面上方。

#### **白名单管理**

输入以逗号分隔的地址进行批量添加或移除。  
可查询某地址是否在白名单。

#### **空投**

向地址列表中每个地址 **空投 1 枚 NFT**。

#### **管理员管理**

添加或移除管理员地址。

#### **合约控制（仅 Owner）**

- 暂停（pause）
- 恢复（unpause）

### 3. 刷新状态

点击“最新の状態を取得”以重新载入链上数据。
