## 概要

本サイトでは、日中学院が発行するデジタルコレクション（NFT）の閲覧・受け取りが可能です。  
管理者は管理画面から各種設定や NFT 管理を行えます。  
MetaMask（ブラウザ拡張機能）を使って接続します。

---

## 事前準備

1. MetaMask ブラウザ拡張機能をインストール（Chrome 推奨）。
2. ウォレットを作成またはインポート。
3. サイトに表示されるコントラクトアドレスが正しいことを確認。
4. MetaMask のネットワークを **Optimism Mainnet** に切り替える。

## 一般利用者（閲覧・受け取り）

### 1. ウォレット接続

画面上部の「MetaMask 接続」をクリック。

### 2. **コレクション一覧**

発行済みのコレクションが表示されます。  
画像が表示されていれば、SVG が正常に登録されています。

### 3. **受け取りページ**

以下の条件を満たすコレクションが表示されます：

- アクティブ状態
- SVG のアップロードが確定済み

#### ボタンが押せない主な理由

- ウォレット未接続 / ネットワーク不一致
- 供給上限に到達
- ホワイトリスト対象外
- すでに受領済み

条件を満たした場合、「受け取る」をクリックし、MetaMask で承認すると **NFT を 1 つ受領**できます。

## 管理者（オーナーまたは管理者権限付きアドレス）

### 1. 管理画面へのアクセス

オーナー/管理者ウォレットで接続すると「管理」タブが表示されます。

### 2. 管理機能

#### **コントラクト情報**

コントラクトアドレス、オーナー、バージョンを表示。

#### **新規作成**

設定項目：

- 名前
- 説明
- 最大供給（0 は無制限）
- ホワイトリスト有効化
- 作成直後にアクティブにするかどうか

#### **SVG アップロード**

オンチェーン SVG をチャンク方式で登録：

1. Token ID、chunkIndex（0 から）、SVG テキスト（約 30 KB 以内推奨）を入力。
2. **ステップ 1：チャンク追加** を実行。
3. 全チャンク追加後、**ステップ 2：アップロード完了（以降は変更不可）** で確定。

#### **状態更新**

Token ID ごとに：

- ホワイトリストの切り替え
- アクティブ / 非アクティブ

現在の状態は画面上部に表示されます。

#### **ホワイトリスト管理**

アドレス（コンマ区切り）を追加/削除。  
対象アドレスの状態確認も可能。

#### **エアドロップ**

指定アドレス一覧へ **1 つずつ配布**。

#### **管理者管理**

管理者アドレスの追加・削除。

#### **コントラクト管理（オーナーのみ）**

- 一時停止（pause）
- 再開（unpause）

### 3. 状態更新

「最新の状態を取得」でオンチェーン情報を再読み込み。
